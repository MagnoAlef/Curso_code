Npm , ou node package manager e um gerenciador de pacotes do Node.js
Obs:Node.js e a ferramenta que permite executar o js fora do navegador
Porque o Npm existe?
    .Os modulos que ja vimos sao muitos uteis para organizar nossas aplicações mas essa nao e sua unica vantagem
    .Com o npm programadores podem  compartilhar seus modulos com outros programadores 
    .O npm e um conjunto de ferramentas que permite compartilhar os modulos na internet, baixar
    modulos para nosso proprio projeto e gerenciar os modulos instalados no nosso projeto
    .Isso tudo aliado ao poder da comunicade open-sourde permite aumentar a nossa produtividade(principio DRY,Don't Repeat Yourself)
O npm tambem pode ser visto como 3 recursos que trabalham juntos
    .1 Uma ferramenta de linha de comando para executar o npm
    .2 Um repositorio online de pacotes onde ficam hospedados os modulos que podemos baixar
    .3 O site https://npmjs.com podemos navegar pelos modulos existentes no repositorio
