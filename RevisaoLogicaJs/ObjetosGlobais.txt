São obejtos padrao da linguagem js que ficam sempre disponiveis para utilizarmos
Atencao:não confundir com o objeto de escopo global window do navegador 
(ou o escopo global de outros ambientes)
Alguns exemplos->
Object -> let pessoa = Object() pessoa -> {}
Array -> let lista = Array() lista -> [] lista = Array("A","b","c") lista = ["A","b","c"]
String -> let a = String() a -> ''
Number -> Number("42") -> 42
Date -> Date() -> Devolver a data atual
Math -> Math.Pi -> 3.14159 -> Math.pow(2,10) -> 1024
Podem ser usados para criar e converter valores
Possuem propriedades e funções uteis e frequentemente necessarios
